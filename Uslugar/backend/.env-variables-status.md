## 📝 Kako Postaviti Stripe Keys

### Opcija 1: Stripe Dashboard (Preporučeno)

1. **Prijavi se na Stripe**: https://dashboard.stripe.com/login
2. **Idi na API Keys**:
   - Developers → API keys
   - Ili direktno: https://dashboard.stripe.com/apikeys
3. **Kopiraj ključeve**:
   - **Secret key**: Klikni "Reveal test key" ili "Reveal live key" i kopiraj
   - **Publishable key**: Već je vidljiv, jednostavno kopiraj
4. **Za Webhook Secret**:
   - Developers → Webhooks → Add endpoint
   - Postavi URL: `https://uslugar-backend.onrender.com/api/payments/webhook`
   - Odaberi events i kopiraj signing secret

### Opcija 2: AWS Secrets Manager (ako imaš pristup)

Ako imaš AWS CLI konfiguriran i pristup Secrets Manager:

```bash
# Stripe Secret Key
aws secretsmanager get-secret-value \
    --secret-id uslugar/stripe-secret-key \
    --region eu-north-1 \
    --query SecretString \
    --output text

# Stripe Publishable Key
```

## 🧾 PDF Fakture – Unicode font (ć, č, ž…)

- **INVOICE_FONT_PATH** ⚠️ – putanja do Unicode TTF fonta koji podržava hrvatske znakove  
  - Preporuka: preuzmi besplatan font, npr.:
    - `DejaVuSans.ttf` (DejaVu Sans)
    - `Roboto-Regular.ttf` (Google Fonts – Roboto)
  - Kopiraj datoteku na server, npr.:
    - Linux: `/opt/uslugar/fonts/DejaVuSans.ttf`
    - Windows: `C:\uslugar-fonts\DejaVuSans.ttf`
  - U `.env` backend-a postavi:
    ```env
    INVOICE_FONT_PATH=/opt/uslugar/fonts/DejaVuSans.ttf
    ```
    ili na Windows serveru:
    ```env
    INVOICE_FONT_PATH=C:\uslugar-fonts\DejaVuSans.ttf
    ```
  - Ako je varijabla postavljena i font postoji, fakture koriste taj font (ispravan prikaz `Rok plaćanja`, `Pretplata`, itd.).
